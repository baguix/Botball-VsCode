## 进阶篇  

演示如下：  
1. 使用“bb”命令连接botball，并使用“bbl”命令列出Botball中的工程。  
![bb and bbl](/show/02.gif)  

2. 使用“bbp”命令编程，代码将直接显示在VSCode中，保存代码将自动更新Wallaby中的代码。  
![bbp](/show/03.gif)  

3. 使用“bbp”命令，新建botball工程。  
![bbp new project](/show/04.gif)  

4. 远程运行botball工程。  
[文件太大，请自行下载演示文件查看](/show/05.gif)
